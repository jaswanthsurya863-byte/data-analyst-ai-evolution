# 📊 AI Impact on Data Analytics Jobs (2024–2036)

> **Are Data Analysts being replaced by AI — or evolving into something more powerful?**
> This project investigates that question using real research, Anthropic's 2026 labor market study, and data from the U.S. Bureau of Labor Statistics.

🌐 **[View Live Project Page](https://jaswanthsurya863-byte.github.io/data-analyst-ai-evolution)**

## 🔍 Project Overview

As AI tools like Claude, ChatGPT, and Copilot rapidly automate data tasks, many are asking: *is the Data Analyst role dying?*

This project analyzes the real impact of AI on data analytics jobs — combining primary research, labor market data, and skills trend analysis to answer:

- Are data analytics jobs **decreasing or increasing**?
- Which analytics **skills are rising** due to AI?
- Which tasks are being **automated**?
- What skills will analysts need by **2030–2036**?

## 📌 Key Findings

### 1. Data Analysts Are NOT Being Replaced — They're Evolving
The role is shifting from **"query executor"** to **"insight steward"** — the human layer that validates AI outputs, applies business context, and connects data to strategy.

### 2. Anthropic's March 2026 Labor Market Report
Anthropic economists studied Claude's actual usage logs across 800+ U.S. occupations and found:

| Metric | Finding |
|--------|---------|
| Theoretical AI task coverage (Computer & Math) | 94% |
| Actual observed AI usage today | 33% |
| Data Analyst theoretical exposure | ~75% |
| Jobs with zero AI exposure | ~30% |
| Unemployment effect so far | ≈ 0 |

> The gap between what AI *could* do and what it *actually does* today — is the analyst's window of opportunity.

### 3. Skills Shifting — Not Disappearing

| Fading Skills | Rising Skills |
|--------------|---------------|
| Writing SQL queries | Validating AI-generated SQL |
| Manual data cleaning | Auditing AI pipelines |
| Building dashboards | Curating context & assumptions |
| Descriptive analytics | Predictive & prescriptive insight |
| Technical execution | Business storytelling |

### 4. Job Growth Is Strong
- 📈 **23% projected growth** in data roles by 2032 (U.S. Bureau of Labor Statistics)
- 🌍 **11.5 million** new data roles expected by late 2026
- 💼 **87%** of analysts report increased strategic importance in 2025
